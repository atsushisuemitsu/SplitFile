# split_file - ファイル分割ツール

100MBを超えるサイズのファイルを100MBごとに分割するツールです。

## 概要

| 項目 | 内容 |
|------|------|
| 分割対象 | 100MB (104,857,600 bytes) を超えるファイル |
| 分割サイズ | 100MB (104,857,600 bytes) |
| 出力先 | 元ファイルと同じディレクトリ |
| 出力ファイル名 | `元ファイル名_part001.拡張子`, `_part002`, ... |

## ファイル構成

| ファイル | 説明 |
|----------|------|
| `split_file.exe` | 実行ファイル（Python不要、単体で動作） |
| `split_file.py` | Pythonソースコード |

## 使い方

### 方法1: ドラッグ＆ドロップ

分割したいファイルを `split_file.exe` にドラッグ＆ドロップしてください。

### 方法2: コマンドライン

```
split_file.exe "C:\path\to\largefile.log"
```

### 引数なしで実行した場合

`split_file.exe` をダブルクリック（引数なし）で実行すると、使用方法を表示して終了します。

## 注意事項

- 100MB以下のファイルを指定した場合は「分割不要」と表示して終了します
- 出力先に同名ファイルが存在する場合は上書きされます
- 分割されたファイルを結合する場合は、コマンドプロンプトで以下を実行してください：
  ```
  copy /b *_part001.LOG + *_part002.LOG + ... 結合後ファイル名.LOG
  ```

## 動作環境

- Windows 10 / 11
- `split_file.exe`: Python不要
- `split_file.py`: Python 3.6以上
